# For My Favorite Human

A premium animated Next.js scrapbook dedicated to a beloved sister.

## Run it

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Make it personal

Open `app/page.tsx` and edit the `memories` and `timeline` arrays near the top. Replace each demo image URL, title, date, and story with your own. The hero copy, sister profile, and letter are in the same file so all the personal writing is easy to find.

## Included

- Responsive photo-story gallery with touch/swipe, thumbnails, autoplay, zoom, keyboard navigation, and modal storytelling
- GSAP scroll reveals and parallax, plus Framer Motion micro-interactions
- Light/dark themes, reduced-motion support, generated music chime, confetti, cat interactions, and letter reveal
- SEO metadata, semantic labels, lazy-loaded gallery images, and static production output
